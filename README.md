# 🌦️ Weather App

A responsive Weather Application built using HTML, CSS, and JavaScript that provides real-time weather information using the OpenWeatherMap API.

---

## 🚀 Features

- 📍 Get weather based on your current location
- 🔍 Search weather by city name
- 🌡️ Display temperature in Celsius
- 💨 Wind speed information
- 💧 Humidity percentage
- ☁️ Cloudiness percentage
- 🏳️ Country flag display
- ⏳ Loading indicator
- 💾 Session Storage support for saved coordinates
- 📱 Responsive UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
- Geolocation API
- Session Storage

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Abhay71044/weather-App.git
```

Move to project directory

```bash
cd weather-App
```

Open `index.html` in your browser.

---

## 🔑 API Setup

Create an account on OpenWeatherMap.

Get your API key.

Replace:

```javascript
const API_KEY = "YOUR_API_KEY";
```

with your own key inside:

```javascript
index.js
```
